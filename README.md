Веб-приложение для создания технического задания на разработку дизайн-проекта, интегрированное с Telegram.

**Основные особенности проекта:**

Технический стек: Next.js 15, React, TypeScript, Tailwind CSS

Интеграция с Telegram: Использует Telegraf для обработки вебхуков от Telegram бота

Форма техзадания: Пошаговая форма с 6 этапами:

Общая информация (имя клиента, адрес, площадь и т.д.)
Информация о резидентах (взрослые/дети, хобби, животные и т.д.)
Помещения (типы, площади)
Демонтажные работы
Информация о строительстве/монтаже
Оборудование и мебель
Генерация PDF: Возможность экспорта заполненного техзадания в PDF-документ

Валидация данных: Использует Zod для валидации форм

Сохранение данных: Использует локальное хранилище для сохранения прогресса

Адаптивный дизайн: Современный UI с использованием компонентов Radix UI

Проект позволяет клиентам заполнять подробное техническое задание для дизайн-проекта через веб-интерфейс, который может быть открыт из Telegram бота, а затем скачивать результат в формате PDF.
